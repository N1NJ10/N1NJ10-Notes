Welcome to the official feedback repository for my notes.

Please note: the actual notes are not hosted in this repository. You can read the published notes here: **https://notes.fadymoheb.com/** .

This repository is only for feedback, corrections, and requests. If you found something useful, noticed a mistake, or want to suggest an improvement, open an issue here.

## How To Use This Repository

Use the **Issues** tab to submit feedback.

- **General Feedback:** Share your thoughts about a note or the overall structure.
- **Correction Request:** Report a typo, broken link, formatting issue, or factual mistake.
- **Addition Suggestion:** Request extra explanation, a missing section, or a completely new topic.

## How To Submit A Request

1. Open the **Issues** tab.
2. Click **New issue**.
3. Choose the template that matches your request.
4. Fill in the requested details, including the note link when possible.
5. Submit the issue.
